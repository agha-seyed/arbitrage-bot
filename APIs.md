# 🌍 لیست APIهای رایگان و اقتصادی برای آربیتراژ ورزشی

همانطور که می‌دانید دریافت ضرایب زنده (Live/Pre-match Odds) یکی از گران‌ترین دیتاهای جهان است. با این حال، چند گزینه رایگان یا بسیار ارزان وجود دارد که می‌توانید در کنار `The-Odds-API` از آن‌ها استفاده کنید.

## ۱. The-Odds-API (توصیه شده - در حال استفاده)
- **آدرس:** `the-odds-api.com`
- **پلن رایگان:** ۵۰۰ ریکوئست در ماه.
- **ویژگی:** بهترین گزینه برای پوشش سایت‌های اروپایی و ایتالیایی. کد فعلی ربات شما بر پایه این API نوشته شده است.

## ۲. API-Football (در بستر RapidAPI)
- **آدرس:** `api-football.com` (یا سرچ در RapidAPI)
- **پلن رایگان:** ۱۰۰ ریکوئست در روز (بسیار عالی برای تست روزانه).
- **ویژگی:** اطلاعات بی‌نظیر فوتبال، لاین‌آپ‌ها و ضرایب پیش‌از‌بازی (Pre-match) از بوکمیکرهای معروف مثل Bet365 و Pinnacle.

## ۳. Pinnacle Sports API
- **آدرس:** `pinnacle.com/en/api`
- **پلن رایگان:** کاملاً رایگان است! اما یک شرط دارد: باید در سایت Pinnacle حساب داشته باشید و حسابتان شارژ شده باشد (حداقل موجودی لازم است).
- **ویژگی:** منبع اصلی ضرایب Sharp در جهان. ترکیب دیتای رایگان Pinnacle با سایت‌های محلی ایتالیا برای آربیتراژ یک استراتژی برنده است.

## ۴. Betfair Exchange API
- **آدرس:** `developer.betfair.com`
- **پلن رایگان:** کلید API برای مصارف شخصی (Personal Use) رایگان است (نیازمند اکانت تایید شده و شارژ شده در Betfair).
- **ویژگی:** دسترسی به بازار تبادل شرط (Exchange). بسیار مناسب برای مدل‌های آربیتراژ Back/Lay.

## ۵. Unofficial APIs (مثل Flashscore یا SofaScore)
- **آدرس:** در پلتفرم `RapidAPI` با جستجوی نامشان پیدا می‌شوند.
- **پلن رایگان:** ۵۰۰ ریکوئست در ماه.
- **ویژگی:** این‌ها دیتای رسمی سایت‌های اسکوربورد را استخراج (Scrape) می‌کنند. سرعتشان ممکن است کمی پایین باشد اما تنوع ورزش‌های بالایی دارند.

> [!TIP]
> **پیشنهاد اجرایی:** 
> بهترین کار برای شروع، ساختن چند اکانت مختلف در The-Odds-API (با ایمیل‌های مختلف) برای داشتن چند API Key ۵۰۰ تایی است تا فعلاً هزینه‌ای پرداخت نکنید!
