# 🏆 Arbitrage Bot 2027 - Italian Arbitrage Beast

## ✨ ویژگی‌ها
- اسکن real-time آربیتراژ
- پشتیبانی از 1X2, Over/Under, BTTS, Asian Handicap
- محاسبه احتمال واقعی + مدل Poisson
- داشبورد وب (FastAPI + React)
- اعلان تلگرام + Voice Alert
- Docker + Multi-Worker
- Machine Learning (XGBoost)
